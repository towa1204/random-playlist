# clasp-typescript-template

GAS 開発用のテンプレートリポジトリ

- TypeScript
- ESLint
- Prettier
- Jest
- @types/google-apps-script

# usage

git clone and npm install

## clasp

clasp はグローバルインストールされていることを想定

`npm install -g @google/clasp`
`clasp login`

## test

jest の使い方は https://typescriptbook.jp/tutorials/jest を参照
